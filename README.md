# Quick Start ✨ 

1. Fork the Project and

```sh
git clone https://github.com/<your_github_username>/Component_DriveToDoor.git
```

2. 

```sh
cd Component_DriveToDoor
```

3. 

```sh
npm install /npm i
```

4.
 ```sh
npm run dev
```
